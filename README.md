# 莉莉安百合风俗店

node.js后端+原生html实现的AI聊天项目，调用大模型API，利用提示词让AI扮演百合女角色，与玩家互动。

## 启动项目

```powershell
cd backend
npm start
```

本地访问 `http://localhost:3000`，即可打开欢迎页面。

## 注册登录

点击首页的**Visit**按钮，或点击下方的任意角色卡，出现登录弹窗。

第一次使用时，需要注册。点击注册按钮，填写用户名和密码，完成注册后即可登录使用。

## 开始对话

登录后，选择想要游玩的角色卡，点击进入聊天页面，开始游戏。

## 新增角色卡

在`data/characters/`文件夹下创建新角色的目录，在里面放入`character.json`（角色的基础信息），`prompt.md`（角色的提示词），以及可选的 `avatar.jpg`（聊天界面的角色头像） 和 `cover.jpg`（角色选择区的大图），即可自动完成角色创建。
